## Required

- You need a 'key.h' file within the same directory as the rest of the contents which will allow you to use the API hooked to the program.

  Example key file:

---------------------------------------------------------------------------------------
```
#ifndef KEY_H
#define KEY_H

// Define your key
#define API_KEY "EXAMPLE_YOUR_API_KEY" // The api key

#endif // KEY_H
```
